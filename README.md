# CH2 계산기 과제

> 필수과제 및 도전과제 최종본은 tag 해놓았으니 참고 바랍니다.

## 1. 필수 과제

step 1 과 step 2로 이루어져 있으며 최종본은 step 2입니다.

#### 1.1 요구사항
- 사용자로부터 두 개의 숫자와 연산자를 입력받아 계산 결과를 출력하는 프로그램을 작성합니다.
- 연산자는 덧셈(+), 뺄셈(-), 곱셈(*), 나눗셈(/)을 지원해야 합니다.
- 사용자가 잘못된 입력을 했을 때 적절한 오류 메시지를 출력해야 합니다.
- 프로그램은 계속해서 사용자로부터 입력을 받아 계산을 수행할 수 있어야 합니다. 사용자가 "exit"를 입력하면 프로그램이 종료됩니다.
